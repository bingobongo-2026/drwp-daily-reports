# 日報マン マーケティング素材

WordPress 固定ページに貼り付けて使える LP（ランディングページ）の素材です。

## 使い方

### 方法 A: そのままカスタム HTML ブロックに貼り付け（推奨）

1. WordPress 管理画面 → **固定ページ → 新規追加**
2. ブロックエディタで **「カスタム HTML」ブロック** を追加
3. [`landing-page.html`](./landing-page.html) の中身を丸ごとコピーして貼り付け
4. プレビューで確認後、必要なリンク先（`href="#"` の部分）を:
   - 「無料で試す」 → 問い合わせフォームページ / Stripe など
   - 「お問い合わせ」 → 同上
   に差し替え
5. 公開

### 方法 B: テーマや別 LP に組み込む

`<style>` ブロックと `<div class="drwp-lp">` 内のマークアップを必要なところに分けてコピーしてください。
すべての CSS は `.drwp-lp` 配下にスコープしているので、サイトの既存スタイルと衝突しません。

## カスタマイズしやすい箇所

| 場所 | 何を変える |
| --- | --- |
| Hero `<h1>` | キャッチコピー（業種・想定顧客に合わせて） |
| Pain points 4 枚 | 自社の顧客のリアルな課題に合わせて文言調整 |
| 料金プランの「お問い合わせ」 | 実際の月額に置換（例 `¥9,800`） |
| FAQ | お客様からよく聞かれる質問に合わせて追加 |
| Hero 背景色 (`#1f2937` / `#2563eb`) | コーポレートカラーに合わせて変更 |
| Pricing の Pro 強調色 (`#8b5cf6`) | ブランドアクセント色に合わせて変更 |

## デザインの方針

- **全インライン CSS**: テーマ依存しない、貼るだけで動く
- **画像ゼロ**: 絵文字 + グラデーションで構築、画像差し替えなしで公開可能
- **`.drwp-lp` スコープ**: サイト本体の CSS と衝突しない
- **レスポンシブ対応**: スマホ幅で自動的にカード縦並びに

## 含まれるセクション

1. **Hero** — メインキャッチ + CTA
2. **Pain points** — 4 つの典型的な課題
3. **3 ステップ** — 入力 → 承認 → 公開のフロー
4. **主要機能** — 9 つの機能カード
5. **AI Pro** — 6 つの AI 機能（紫グラデーション）
6. **セキュリティ** — Ed25519、法令対応、権限分離など
7. **料金プラン** — Basic / Pro の 2 段構成
8. **FAQ** — 7 件のよくある質問
9. **Final CTA** — 黒背景の最終アクション喚起
