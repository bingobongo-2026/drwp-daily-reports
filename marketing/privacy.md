### 3.3 利用者の WordPress サイト上に保存される情報 (お預かりではないもの)

利用者が本プラグインを通じて作成・編集する以下のデータは、**利用者自身が運営する WordPress サイトのデータベースに保存** されるものであり、原則として当社は内容を取得・閲覧しません。

- 日報本文、特記事項、次回予定
- 写真ファイル及び付随するキャプション
- 案件 (顧客) 情報、社員情報、コメント、レビュー履歴

ただし、利用者から障害調査・サポートのために明示的に開示の依頼があった場合に限り、必要な範囲で当社スタッフが当該データを参照することがあります。

## 4. 利用目的

当社は、取得した個人情報を次の目的の範囲内でのみ利用します。

1. 本サービスの提供・運営 (アカウント認証、ライセンス検証、AI 中継、サポート対応等)
2. サブスクリプション契約に基づく課金・請求・領収書発行
3. サービス改善のための統計分析 (個人を識別しない形に加工)
4. 重要なお知らせ (規約改定、緊急障害通知、セキュリティ告知等) の送付
5. 不正利用・不正アクセスの検知・防止
6. 法令に基づく対応、捜査機関・公的機関からの正当な要請への対応

メールマガジン・営業案内等のマーケティング目的での利用は、別途利用者から同意を得た場合に限ります。

## 5. 第三者提供

当社は、次のいずれかに該当する場合を除き、利用者の同意なく個人情報を第三者に提供しません。

1. 法令に基づく場合
2. 人の生命、身体又は財産の保護に必要で本人の同意を得ることが困難な場合
3. 公衆衛生の向上又は児童の健全な育成の推進のために特に必要で本人の同意を得ることが困難な場合
4. 国の機関若しくは地方公共団体又はその委託を受けた者が法令の定める事務を遂行することに対して協力する必要がある場合
5. 事業承継 (合併・会社分割・事業譲渡等) に伴い個人情報を移転する場合

### 6.1 AI 機能利用時の特別な注意事項

「運営契約 API を使う」モードを選択した利用者が AI 機能を利用すると、生成 AI への入力テキスト (例: 日報の本文、案件名、特記事項等) は当社のライセンスサーバを経由して **米国の Anthropic, PBC** に送信されます。

- 送信内容: 利用者が AI 機能ボタンから選択した処理 (要約・下書き生成等) の入力データ
- 送信されないもの: アカウント情報、写真ファイル、利用者の連絡先
- 当社サーバでの保存: 呼び出し時刻と回数のみカウンタとして保持し、本文・回答は保存しません
- Anthropic 側の取り扱い: Anthropic は当該データを Anthropic API の利用規約 (https://www.anthropic.com/legal) に従って取り扱い、API 経由のデータは原則としてモデル学習には使用しません

「自分の API キーを使う」モードを選択した場合は、当社サーバを経由せず利用者自身が契約した AI プロバイダへ直接送信されます。

### 6.2 個人情報の国外移転についての同意

決済情報の取扱いに伴いアメリカ合衆国 (Stripe) へ、AI 機能の利用に伴いアメリカ合衆国 (Anthropic) へ、それぞれ個人情報が移転される場合があります。利用者は、本ポリシーへの同意又は AI 機能の有効化をもって、当該移転に同意したものとみなします。

各国の個人情報保護に関する制度は個人情報保護委員会のウェブサイト (https://www.ppc.go.jp/) でご確認いただけます。

## 7. Cookie 等の利用

当社の管理画面・ライセンス管理ページでは、利用者の利便性向上と不正アクセス対策のため Cookie 及びローカルストレージを利用します。

- 必須 Cookie: ログインセッションの維持、CSRF 対策トークン、2 要素認証セッション
- 分析 Cookie: 利用していません (将来導入する場合は本ポリシー改定時に明記します)

ブラウザの設定で Cookie を無効にできますが、その場合は管理画面にログインできなくなります。

## 8. 安全管理措置

当社は、個人情報の漏えい・滅失・毀損の防止のため、合理的な技術的・組織的措置を講じます。

- 通信の暗号化: ライセンスサーバとの全通信を TLS で暗号化、Let's Encrypt 証明書を自動更新
- 認証: 管理画面は HTTP Basic 認証 + TOTP (RFC 6238) による 2 要素認証
- アクセス制御: 連続失敗時の IP 単位レート制限 (Fail2ban 相当)
- 監査ログ: 管理画面ログイン、署名鍵ローテート、TOTP 設定変更等を 90 日以上保管
- 署名: ライセンス検証レスポンスは Ed25519 で署名し、改ざんを防止
- バックアップ: データベース・署名鍵を毎日自動バックアップ (14 日分の世代保持)
- アクセス権限: 個人情報を取り扱う従業員を最小限に限定し、退職時には速やかに権限を剥奪

サブスクリプションを解約した場合の取扱いは次のとおりです。

- ライセンスサーバ上の契約情報: 解約後 90 日間は復旧用に保持し、その後削除します
