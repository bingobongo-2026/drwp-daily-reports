3 ステップで日報が公開記事になる

入力 → 承認 → 公開、現場と事務所の役割を分けて運用

1

📱 現場から日報を送る

スマホで写真を撮って、作業内容を入力。Before / After のラベルも付けられます。送信先は「レビュー待ち」へ。

2

✅ 事務所がレビュー

表現の調整やコメントを付けて「承認」または「差戻し」。承認後は公開記事化のステージへ進みます。

3

📰 ワンクリックで公開

テンプレートを選んで「記事を作成」。AI がタイトルから本文まで整え、WordPress 投稿として公開できます。

主要機能

運用に必要なものを 1 プラグインに集約

📱

スマホ最適化フォーム

現場から片手で日報を送れる入力 UI。写真のドラッグ並び替え、キャプション、Before / After 指定対応。

✅

レビューワークフロー

レビュー待ち / 承認 / 差戻し / 編集依頼の 4 状態。コメントで現場とやり取りでき、操作はすべて履歴に残ります。

📰

3 つの記事テンプレート

「標準」「案件レポート」「ビフォーアフター」の 3 種類。写真の振り分けや案件情報の見せ方を切替できます。

📅

カレンダー × リスト

月単位カレンダーで「いつ何があったか」、リストで「内容のスキャン」。状態色で一目で進捗が分かります。

🏢

顧客 / 案件マスタ

顧客 → 案件 → 日報 → 記事 の一気通貫管理。グループでタグ付け、絞り込みも自由自在。

📆

予定（プラン）機能

次回訪問予定をカレンダーに登録。チップをタップすれば、その予定から日報をワンタップで起こせます。

📊

CSV エクスポート

絞り込んだ日報を Shift-JIS で一括出力。Excel 日本語版でそのまま開けます。

🗄️

アーカイブ / 法定保存

建設業法対応。日報のハード削除はなし、論理アーカイブ → 30 日後に管理者のみ完全削除可能。

📜

操作履歴 (監査ログ)

「誰が・いつ・何をしたか」を全イベント記録。自動保存期間設定で長期保持にも対応。

🤖 AI が記事化と運用を後押し

OpenAI / Anthropic の API キーで動作。Pro プランの強み。

✨

公開記事の下書き生成

「何を / どこで / いつ」が明確なタイトル、導入文、本文、今後の予定をワンクリック生成。

🧭

次回訪問ブリーフィング

過去の日報を読み込み、次の訪問で持っていくべき情報・確認事項を要約します。

📈

案件サマリ

月次・四半期で案件の進捗、気付き、課題を AI がまとめて報告書化。

⚠️

対応必須アラート

特記事項から「これは要対応」を抽出。ダッシュボードに警告を表示します。

🔍

振り返りアドバイス

絞り込み中の日報を分析、過去の成功例 / 失敗例から今後の向き合い方を提案。

🔐

個人情報の自動検知 + マスク

公開前に顧客名・電話・メール・郵便番号を検知。ワンクリックで「Y様」「○○○-○○○○-○○○○」にマスク。

監査ログの保存期間を設定可能。日次 cron で古い行を自動削除し、DB を肥大化させません。

よくある質問

既存の WordPress サイトに導入できますか？
はい、プラグインを有効化するだけで、既存サイトに日報マンの管理メニューと WordPress 投稿として記事が出るようになります。テーマは選びません。
AI 機能には別途料金がかかりますか？
OpenAI / Anthropic の API キーを「AI 設定」に入力していただきます。プロバイダ側の従量課金（記事 1 本生成あたり数円程度）が別途発生します。
現場の社員はスマホで日報を送れますか？
はい。専用のモバイル最適化フォームがあり、写真撮影・添付・送信まで片手で完結します。送信した日報は事務所側のレビュー待ちに入ります。
個人情報の取り扱いはどうなっていますか？
公開記事化前に顧客名・電話番号・メール・郵便番号を自動検知して警告します。ワンクリックで「Y様」「○○○-○○○○-○○○○」のような形式に自動マスクできます。
過去の日報を消したい場合はどうすれば？
建設業法の保存義務と内部不正対策の観点から、ハード削除は管理者のみ・アーカイブ後 30 日経過後のみ可能です。それまでは「アーカイブ」で一覧から非表示にできます。
ライセンス検証はどう動きますか？
独立したライセンスサーバーが Ed25519 で署名応答を返し、プラグイン側で libsodium 検証します。応答を改ざんしても機能解放にはつながりません。
無料体験はできますか？
はい。お問い合わせフォームから無料体験ライセンスをお申し込みください。すぐに発行してメールでお送りします。
