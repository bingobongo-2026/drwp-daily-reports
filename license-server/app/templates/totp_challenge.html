{% endblock %} {% block help_title %}2FA 認証について{% endblock %} {% block help_body %}

このサーバは 2 段階認証 (2FA) が有効です。Basic 認証 (ユーザー名+パスワード) に加えて、Authenticator アプリの 6 桁コードが必要です。

Authenticator を紛失した場合は、セットアップ時に保管したリカバリーコード (XXXX-XXXX 形式) を代わりに入力してください。リカバリーコードは 1 回ずつ使い切りです。

緊急時の解除手段: コンテナの環境変数 DRWP_TOTP_DISABLED=1 を設定してサーバを再起動すると、2FA ゲートが一時的にオフになります。ログインしたら直ちに 2FA を無効化 → 再セットアップしてください。
