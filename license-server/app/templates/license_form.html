各項目の入力ルール

  • ライセンスキー — 任意の文字列。重複不可。人間が読める接頭辞+ランダム ID (例 ACME-7F2C-9KQ4) がおすすめ。{% if license %}編集時は変更できません。{% endif %}
  • ドメイン — プラグインが動く WordPress サイトのホスト名。https:// や末尾の / は付けません。プラグインは wp_parse_url(home_url(), PHP_URL_HOST) の結果を送るため、WP の「サイトアドレス」設定と一致させてください。www. の有無も完全一致である必要があります。
  • プラン — standard / pro から選択。ロジックには使われず、画面表示用です。
  • 状態 — active が有効、inactive はプラグイン側で書き込み・記事化がブロックされます。一時停止に使えます。
  • 有効期限 — 空欄は無期限。日時を入れる場合は ISO 8601 形式 (YYYY-MM-DDTHH:MM:SS+00:00) で。タイムゾーン省略は避けてください。

こんなときは

  • 失効させたい — 削除ではなく、編集で状態を inactive にする。履歴が残り、復活も簡単です。
  • ドメインが変わった — 編集でドメイン欄を新しい値に更新。
  • キーを変えたい — このフォームでは変更できないので、削除→新規作成です。

保存後にやること

WordPress 側で「ライセンスをチェック」を実行して、状態が反映されることを確認してください。古い状態がキャッシュされていることがあります。
