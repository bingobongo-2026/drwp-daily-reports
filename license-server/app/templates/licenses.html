この画面でできること

登録済みのライセンスを一覧表示し、新規発行・編集・削除を行えます。

各カラムの意味

  • キー — WordPress プラグインの「ライセンスキー」欄に入れる値です。
  • ドメイン — このキーを使ってよいサイトのホスト名 (例 example.com)。プラグインから送られてくる値と完全一致する必要があります。
  • プラン — standard や pro など任意のラベル。表示用です。
  • 状態 — active 以外はプラグイン側で書き込み・記事化がブロックされます。
  • 有効期限 — ISO 8601 形式の日時。— は無期限。

操作

  • 新規作成 — 右上のボタンから。ライセンスキーは重複不可。
  • 編集 — 行の「編集」ボタン。license_key は変更できません。変えたい場合は削除→作成。
  • 削除 — 行の「削除」ボタン。取り消せません。一時的に止めたいだけなら、削除よりも編集で状態を inactive にする方が安全です。

WordPress プラグイン側との連携

新しくライセンスを作ったら、WordPress 管理画面の「DRWP Daily Reports → ライセンス」ページで、ここで作った license_key を入力し「公開鍵を取得」→「ライセンスをチェック」を実行してください。状態が active、署名検証が valid になれば連携完了です。
